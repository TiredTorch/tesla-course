# Car viewer

The project-task for three js course

## How to run

To load project repository run:
### `git clone https://github.com/TiredTorch/tesla-course.git`

After that, in the project directory, you should run:

### `npm start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
